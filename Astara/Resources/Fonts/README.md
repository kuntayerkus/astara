# Astara Fonts

Download the following font files and place them in this directory.

## Cormorant Garamond (Headers/Titles)
Download from: https://fonts.google.com/specimen/Cormorant+Garamond

Required files:
- CormorantGaramond-Bold.ttf
- CormorantGaramond-SemiBold.ttf
- CormorantGaramond-Medium.ttf
- CormorantGaramond-Regular.ttf

## Plus Jakarta Sans (Body/Labels)
Download from: https://fonts.google.com/specimen/Plus+Jakarta+Sans

Required files:
- PlusJakartaSans-Bold.ttf
- PlusJakartaSans-SemiBold.ttf
- PlusJakartaSans-Medium.ttf
- PlusJakartaSans-Regular.ttf

## Setup
These fonts are registered in the project via `project.yml` Info.plist `UIAppFonts` array.
No additional configuration needed after placing the .ttf files here.
